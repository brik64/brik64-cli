## 0.1.0-beta.15.7

### Fixed

- Certification now rejects return type mismatches, implicit numeric coercions, missing PCD headers, reserved identifiers, and non-exhaustive return paths.
- TypeScript and Python generated code now use deterministic integer division helpers for integer return types, matching Rust golden vectors.

### Changed

- Lift previews now fail closed or report warning codes when numeric precision or unsupported constructs would lose semantics.
- PCD files generated by templates and migrations use the brik64.pcd_file.v1 header contract.
- The installable CLI package includes the offline local runtime bundle and release-generation evidence under conservative public claim boundaries.
